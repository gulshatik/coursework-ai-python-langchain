# Создание простого AI‑агента на Python с использованием LangChain

> **Цель** – написать скрипт, реализующий иерархического AI‑агента, который умеет планировать список покупок:  
> 1️⃣ Принимать от пользователя перечень продуктов.  
> 2️⃣ Для каждого продукта запрашивать примерную цену (через субагент).  
> 3️⃣ Суммировать итоговую стоимость корзины.

---

## 📦 Стек технологий

| Технология | Версия / Пакет |
|------------|----------------|
| Python     | ≥ 3.10         |
| LangChain  | `1.2.10`       |
| langchain‑openai | `1.1.9`   |
| LM Studio (локальный сервер LLM) | любой, доступный по `http://localhost:1234/v1` |

> **Важно** – все зависимости устанавливаются через `pip install`.

## 🧩 Архитектура решения

| Компонент | Что делает |
|-----------|------------|
| **LLM**   | Подключается к локальному серверу LM Studio через `ChatOpenAI`. |
| **Инструмент `get_price`** | Принимает `product` и `city`, внутри создает субагента, который генерирует таблицу цен. |
| **Субагент** | Использует тот же LLM, но без внешних инструментов; формирует ответ в виде Markdown‑таблицы. |
| **Главный агент** | Имеет системный промпт «Ты помощник по планированию покупок» и использует `get_price`. |
| **Запрос** | Пользовательский ввод: *«Помоги составить список покупок: молоко, хлеб, яблоки. Я нахожусь в Казани.»* |

## 📄 Полный код (`main.py`)

```python
#!/usr/bin/env python3
# -*- coding: utf-8 -*-

"""
main.py – пример иерархического AI‑агента на LangChain.
Запуск:
    python main.py
"""

from __future__ import annotations

import json
import sys
from typing import Any, Dict, List

from langchain.agents import create_agent
from langchain.tools import tool
from langchain_openai import ChatOpenAI
from pydantic import BaseModel, SecretStr


# --------------------------------------------------------------------------- #
# 1️⃣ Подключение к локальной модели LLM
# --------------------------------------------------------------------------- #

LLM_MODEL_NAME = "my-local-model"  # <-- замените на название вашей модели в LM Studio

llm = ChatOpenAI(
    model=LLM_MODEL_NAME,
    base_url="http://localhost:1234/v1",
    api_key=SecretStr("fake"),
    temperature=0.7,
)


# --------------------------------------------------------------------------- #
# 2️⃣ Инструмент get_price(product, city) → str
# --------------------------------------------------------------------------- #

class PriceArgs(BaseModel):
    """Аргументы инструмента get_price."""
    product: str
    city: str


@tool
def get_price(product: str, city: str) -> str:
    """
    Получить примерную цену продукта в указанном городе.

    Возвращает Markdown‑таблицу вида:

        | Продукт | Цена (руб.) | Магазин |
        |---------|-------------|---------|
        | Молоко  | 89          | Магнит  |

    Субагент генерирует таблицу, опираясь на исторические данные о ценах.
    """
    # Создаём субагента без внешних инструментов
    price_agent = create_agent(
        model=llm,
        tools=[],  # субагент сам не использует инструменты
        system_prompt=f"""
Генерируй таблицу цен для продукта "{product}" в городе "{city}".  
Таблица должна содержать три колонки: Продукт, Цена (руб.), Магазин.  
Ответ должен быть только Markdown‑таблицей без лишних комментариев.
""",
    )

    # Запускаем субагент
    response = price_agent.invoke(
        {
            "messages": [
                {"role": "human", "content": f"Сгенерируй таблицу цен для {product} в городе {city}."}
            ]
        }
    )

    # Извлекаем текст ответа (последнее сообщение)
    last_msg = response["messages"][-1]
    return last_msg.content.strip()


# --------------------------------------------------------------------------- #
# 3️⃣ Главный агент
# --------------------------------------------------------------------------- #

main_agent = create_agent(
    model=llm,
    tools=[get_price],  # подключаем инструмент get_price
    system_prompt="Ты помощник по планированию покупок.",
)


# --------------------------------------------------------------------------- #
# 4️⃣ Запрос и вывод результата
# --------------------------------------------------------------------------- #

def format_message(msg: Dict[str, Any]) -> str:
    """
    Преобразует сообщение из агента в читаемый формат.
    Если сообщение содержит tool_calls – выводим вызов инструмента,
    иначе печатаем текст.
    """
    if "content" in msg and msg["content"]:
        return msg["content"].strip()
    # Вызовы инструментов
    calls = msg.get("tool_calls", [])
    if not calls:
        return ""
    parts = []
    for call in calls:
        name = call["name"]
        args = json.dumps(call["args"], ensure_ascii=False)
        parts.append(f"{name}({args})")
    return "\n".join(parts)


def main() -> None:
    user_query = (
        "Помоги составить список покупок: молоко, хлеб, яблоки. Я нахожусь в Казани."
    )

    # Вызов главного агента
    result = main_agent.invoke(
        {
            "messages": [
                {"role": "human", "content": user_query}
            ]
        }
    )

    # Вывод всех сообщений
    print("\n--- Итерация вызовов и ответ ---\n")
    for msg in result["messages"]:
        formatted = format_message(msg)
        if formatted:
            print(formatted + "\n")

    # Финальный вывод (последнее сообщение)
    final_msg = result["messages"][-1]
    print("\n=== Финальный ответ ===")
    print(final_msg.content.strip())


if __name__ == "__main__":
    try:
        main()
    except KeyboardInterrupt:
        sys.exit(0)
```

---

## 📄 Как это работает

1. **LLM** подключается к LM Studio через `ChatOpenAI`.  
2. При вызове `get_price` создаётся субагент, который генерирует таблицу цен в Markdown‑формате.  
3. Главный агент использует этот инструмент для каждого продукта из запроса пользователя.  
4. В конце выводятся все промежуточные вызовы и финальный ответ с суммой.

---

## 📦 Как запустить

1. Убедитесь, что LM Studio запущен на `http://localhost:1234/v1`.  
2. Запустите скрипт:

```bash
python main.py
```

Вы увидите цепочку вызовов инструментов и итоговый ответ в консоли.
