# Создать простого AI‑агента на Python с использованием LangChain

## Задание

Написать скрипт `main.py`, реализующий **иерархического AI‑агента** для планирования списка покупок, используя библиотеку LangChain и локальную языковую модель (LM Studio).

> **Важно:** Весь код должен быть помещён в один файл `main.py`.  
> Запуск осуществляется командой:
> ```bash
> python main.py
> ```

---

## Что нужно сделать

1. **Подключиться к локальной модели** через OpenAI‑совместимый API.
2. Создать инструмент `get_price(product: str, city: str) -> str` с декоратором `@tool`.  
   Внутри инструмента реализовать субагент, который генерирует реалистичную цену продукта в указанном городе и возвращает таблицу:
   ```text
   | Продукт | Цена (руб.) | Магазин |
   ```
3. Создать главный агент с системным промптом `Ты помощник по планированию покупок` и передать ему инструмент `get_price`.
4. Вызвать агента запросом:  
   ```
   Помоги составить список покупок: молоко, хлеб, яблоки. Я нахожусь в Казани.
   ```
5. Вывести все сообщения из ответа (включая вызовы инструментов и финальный ответ) в читаемом виде.

---

## Полный код (`main.py`)

```python
#!/usr/bin/env python3
# -*- coding: utf-8 -*-

"""
Иерархический AI‑агент для планирования списка покупок.
"""

from __future__ import annotations

import json
import os
import sys
from typing import Any, Dict, List

from langchain.agents import create_agent
from langchain_openai import ChatOpenAI
from pydantic import SecretStr
from langchain.tools import tool


# --------------------------------------------------------------------------- #
# 1. Настройка LLM
# --------------------------------------------------------------------------- #

MODEL_NAME = os.getenv("LLM_MODEL", "gpt-4o-mini")  # замените на название модели в LM Studio

llm = ChatOpenAI(
    model=MODEL_NAME,
    base_url="http://localhost:1234/v1",
    api_key=SecretStr("fake"),
    temperature=0.7,
)


# --------------------------------------------------------------------------- #
# 2. Инструмент get_price
# --------------------------------------------------------------------------- #

@tool
def get_price(product: str, city: str) -> str:
    """
    Возвращает таблицу с ценой продукта в указанном городе.

    Пример ответа:
        | Продукт | Цена (руб.) | Магазин |
        |---------|-------------|---------|
        | Молоко  | 89          | Магнит  |

    Параметры
    ----------
    product : str
        Название продукта.
    city : str
        Город, в котором нужно узнать цену.

    Returns
    -------
    str
        Таблица с ценой и магазином.
    """
    # Создаём субагент без внешних инструментов
    sub_agent = create_agent(
        model=llm,
        tools=[],
        system_prompt=f"""
            Ты эксперт по ценам в {city}.  
            Сгенерируй реалистичную цену на продукт «{product}» в этом городе.  
            Ответ должен быть таблицей формата:
            
            | Продукт | Цена (руб.) | Магазин |
            |---------|-------------|---------|
            | …       | …           | …       |
        """,
    )

    # Запускаем субагент без дополнительного контекста
    sub_response = sub_agent.invoke(
        {"messages": [{"role": "human", "content": ""}]}
    )
    # Возвращаем только последнее сообщение (таблицу)
    return sub_response["messages"][-1]["content"].strip()


# --------------------------------------------------------------------------- #
# 3. Главный агент
# --------------------------------------------------------------------------- #

main_agent = create_agent(
    model=llm,
    tools=[get_price],
    system_prompt="Ты помощник по планированию покупок.",
)


# --------------------------------------------------------------------------- #
# 4. Вызов агента и вывод результата
# --------------------------------------------------------------------------- #

def format_message(msg: Dict[str, Any]) -> str:
    """
    Преобразует сообщение из цепочки в читаемый формат.
    """
    if "content" in msg and msg["content"]:
        return msg["content"]
    # Если сообщение содержит вызов инструмента
    tool_calls = msg.get("tool_calls", [])
    if not tool_calls:
        return ""
    call = tool_calls[0]
    name = call.get("name")
    args_json = call.get("args") or "{}"
    try:
        args = json.loads(args_json)
    except Exception:
        args = {}
    args_str = ", ".join(f"{k}={v!r}" for k, v in args.items())
    return f"{name}({args_str})"


def main() -> None:
    user_query = (
        "Помоги составить список покупок: молоко, хлеб, яблоки. Я нахожусь в Казани."
    )
    response = main_agent.invoke(
        {"messages": [{"role": "human", "content": user_query}]}
    )

    print("\n--- Итоговый вывод ---\n")
    for msg in response["messages"]:
        print(format_message(msg))
        print("-" * 40)


if __name__ == "__main__":
    try:
        main()
    except KeyboardInterrupt:
        sys.exit(0)
```

---

## Как это работает

1. **LLM** подключается к локальному серверу LM Studio через `ChatOpenAI`.
2. Инструмент `get_price` создаёт субагент, который генерирует таблицу с ценой продукта в заданном городе.
3. Главный агент использует этот инструмент для каждого продукта из запроса пользователя и суммирует итоговую стоимость.
4. В консоли выводятся все шаги: вызовы инструмента и финальный ответ.

## Что проверить

- После запуска `python main.py` вы должны увидеть цепочку сообщений, где каждый продукт обрабатывается через `get_price`, а затем выводится итоговый список с суммой.
- Если модель не отвечает корректно, убедитесь, что LM Studio запущен и доступен по адресу `http://localhost:1234/v1`.

## Возможные улучшения

| Идея | Как реализовать |
|------|-----------------|
| **Кеширование цен** | Сохранять результаты `get_price` в словарь, чтобы не запрашивать цену одного и того же продукта несколько раз. |
| **Поддержка множественных городов** | Добавить параметр `city` в запрос пользователя и передавать его в инструмент. |
| **Расчёт скидок/акций** | Расширить субагент, чтобы он учитывал сезонные акции. |
